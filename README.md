This repository hosts `aihorde_net.py`, a simple module written by my using Python.
It enables developers to interact with the AI Horde.
You may visit the AI Horde's [official webpage](https://aihorde.net/),
and or try out its [text-to-text](https://lite.koboldai.net/) and [text-to-image](https://tinybots.net/artbot/create) functionality through official services.
